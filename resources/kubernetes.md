## CheatSheet

### Minikube

* Minikube version: `minikube version`
* Start cluster: `minikube start`
* Delete cluster: `minikube delete`

### Kubernetes

* Cluster version: `kubectl version`
* Cluster information: `kubectl cluster-info`
* List nodes: `kubectl get nodes`
